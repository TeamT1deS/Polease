# Polease

## Project setup (Recommend use Yarn)
```
yarn config set electron_mirror https://npmmirror.com/mirrors/electron/
# 设置 Electron 二进制文件的下载镜像 (国内推荐使用 npmmirror)
# Set Electron binary file download mirror (recommend npmmirror)

yarn install
# 安装依赖
```
